# emb
emb - "Embedded development boards for the AI era"

emb is an AI-native embedded development framework.
It closes the loop between code generation, flashing, hardware observation, and AI feedback.

emb frees you from tedious tester work during your valuable time off so you can focus on coming up with ideas.☕


## Install

```bash
pip install emb


## Usage
emb flash attiny202 examples/attiny202/blink.c COM3
